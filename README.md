## Tutorial
cd /etc/ansible/roles/ && git clone https://github.com/88K/tutorials.git && ansible-playbook install.yaml

### Applications: 
Grafana (Port: 443), Node-Exporter (Port: 9100), HA-Proxy (Port: 80 > Port: 443)

### SystemD Services:
grafana, node-exporter, haproxy

### Install Location:
/opt/[application]
